## 4. Presentación del proyecto

**FixFlow** es una aplicación de escritorio desarrollada en Java con JavaFX para la gestión de activos e incidencias técnicas en empresas.

**Problema que resuelve:** en muchas empresas, la gestión de averías se hace de forma manual o con herramientas genéricas. FixFlow centraliza el proceso: registrar activos, reportar incidencias, asignarlas a técnicos y dejar constancia de cada resolución.

**Para quién está pensado:** empresas medianas con equipos técnicos de mantenimiento, como instalaciones industriales, edificios de oficinas o centros educativos.

**Tecnologías utilizadas:** Java 17, JavaFX, MySQL, JDBC, Maven, CSS personalizado.

**Qué he aprendido desarrollándolo:**
- Cómo conectar una aplicación Java con una base de datos real mediante JDBC.
- Diseño de interfaces atractivas con CSS en JavaFX.
- Arquitectura en capas MVC aplicada a un proyecto real.
- Control de versiones con Git y GitHub desde el inicio del proyecto.
- Gestión de roles y permisos en una aplicación de escritorio.

## 6. Reflexión personal

**Qué he aprendido:** a desarrollar una aplicación completa de principio a fin, conectada a una base de datos real, con interfaz gráfica y sistema de usuarios. También he aprendido a usar Git de forma continuada y a organizar un proyecto en capas.

**Qué se me ha dado mejor:** el diseño visual de la interfaz. Disfruto ajustando los estilos y consiguiendo que la aplicación tenga un aspecto profesional y coherente.

**Qué me ha costado más:** conectar correctamente las vistas con los datos de la base de datos, especialmente los JOIN entre tablas y hacer que los datos aparecieran correctamente en las TableView de JavaFX.

**Qué mejoraría:** añadiría un sistema de búsqueda y filtrado dentro de las tablas, y me gustaría explorar cómo convertir esta aplicación en una versión web usando Spring Boot y React en el futuro.

**Pasos que quiero dar:** aprender desarrollo web frontend con HTML, CSS y JavaScript durante el verano, y en 2º profundizar en Spring Boot para el backend. También quiero trabajar más mi perfil de GitHub y LinkedIn con proyectos propios.
